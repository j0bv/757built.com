757Built API Documentation

Introduction

The 757Built API provides access to technological development information across the Hampton Roads region. This API allows developers to access data about research projects, technological advancements, and innovation across the region.

Base URL

https://757built.com/api

Authentication

The API is currently publicly accessible without authentication.

Rate Limiting

API requests are limited to 100 requests per hour per IP address.

Endpoints

GET /technologies

Returns a list of all technology areas in the Hampton Roads region.

Example Request

GET https://757built.com/api/technologies

Example Response

{
  "status": "success",
  "count": 2,
  "data": [
    {
      "id": 1,
      "name": "Quantum Computing in Hampton Roads",
      "slug": "quantum-computing-in-hampton-roads",
      "description": "Research and development in quantum computing technologies."
    },
    {
      "id": 2,
      "name": "Renewable Energy Projects in Virginia Beach",
      "slug": "renewable-energy-projects-in-virginia-beach",
      "description": "Sustainable energy initiatives in Virginia Beach area."
    }
  ]
}

GET /technologies/{slug}

Returns detailed information about a specific technology area.

Parameters

  • slug - The slug identifier for the technology area

Example Request

GET https://757built.com/api/technologies/quantum-computing-in-hampton-roads

Example Response

{
