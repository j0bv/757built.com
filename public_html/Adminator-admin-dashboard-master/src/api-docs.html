757Built API Documentation

Access technology development information for the Hampton Roads region.

Base URL

https://757built.com/api

Available Endpoints

GET /technologies

Returns a list of all technology areas in Hampton Roads.

Example Request:

GET https://757built.com/api/technologies

Example Response:

{
  "status": "success",
  "count": 3,
  "data": [
    {
      "id": 1,
      "name": "Quantum Computing in Hampton Roads",
      "slug": "quantum-computing-in-hampton-roads",
      "description": "Research and development in quantum computing."
    },
    {
      "id": 2,
      "name": "Renewable Energy in Virginia Beach",
      "slug": "renewable-energy-in-virginia-beach",
      "description": "Clean energy initiatives and research."
    }
  ]
}

GET /technologies/{slug}

Returns detailed information about a specific technology area.

Parameters:

  • slug - The slug identifier for the technology area

Example Request:

GET https://757built.com/api/technologies/quantum-computing-in-hampton-roads

Example Response:

{
  "status": "success",
  "data": {
    "id": 1,
    "technology_area_id": 1,
    "key_players": "Jefferson Lab, ODU Physics Department",
    "technological_development": "5-qubit quantum processor development",
    "project_cost": "$4.3 million (NSF Grant)",
    "information_date": "2024-03-15",
    "event_location": "Newport News, VA",
    "contact_information": "quantum@jlab.org",
    "number_of_sources": 3,
    "consolidation_date": "2024-04-20T08:15:30",
    "technology_area_name": "Quantum Computing in Hampton Roads"
  }
}

Rate Limiting

The API is limited to 100 requests per hour per IP address.

Rate limit headers are included in all responses:

  • X-RateLimit-Limit: Maximum requests per hour
  • X-RateLimit-Remaining: Remaining requests in the current hour
  • X-RateLimit-Reset: Seconds until the rate limit resets

Errors

All errors return a JSON response with a status and message:

{
  "status": "error",
  "message": "Error description"
}
